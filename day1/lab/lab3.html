Why Your Results May Differ From Your Classmate's

Even with the same prompt and same model version, two students can get different outputs. Here is why — and why it matters for your career.

🎲
Temperature — Built-in Randomness
Every LLM has a temperature setting that controls how much randomness is in each response. The default is not zero. So the same prompt, same model, same second — can produce different words. This is by design, not a bug. It makes responses feel natural rather than robotic.
🧠
Different Training, Different Personality
ChatGPT, Claude, and Gemini were each trained on different datasets with different human feedback (RLHF). This gives each model a distinct "personality" — Claude tends to be more careful and hedged, ChatGPT more direct, Gemini more structured. Same question, different character.
🗂️
Your Personal Context
ChatGPT's Memory feature stores facts about you across conversations — your name, interests, past topics. Claude remembers within a session. If two students have different stored context, the same prompt gives different answers. This is why Step 2 above (disable memory) matters for a fair comparison.
Enterprise takeaway: This is exactly why production AI systems use API endpoints, not personal chat accounts. The API lets engineers set temperature = 0 for reproducible outputs, pass a controlled system prompt every time, and guarantee zero personal memory. What you are seeing today is the uncontrolled version — the personal account. Real MNC deployments engineer all of this out.
