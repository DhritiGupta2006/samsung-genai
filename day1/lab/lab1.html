3
Verify Gemini Pro access
Open Gemini using your student Google account and confirm Pro tier is active.
Open Gemini ↗
Check the model selector at the top of the page. You should see Gemini 1.5 Pro or higher. If you only see Gemini Flash, use the dropdown to switch to Pro.
4
Fork the training repository
Go to the trainer's GitHub repository and fork it. All your lab outputs will be pushed to your fork.
Open Trainer Repo ↗
The trainer will share the exact GitHub link on the projector. Open it in your browser.
Click Fork (top-right corner on GitHub) → leave all defaults → click Create fork.
Your fork URL will be: github.com/YOUR-USERNAME/samsung-genai
5
Clone your fork to this machine
You will use Git Bash to download the repo to your computer.
How to open Git Bash: Press the Windows key, type Git Bash, press Enter.
A black terminal window will open. That is where you type the commands below.
# Replace YOUR-USERNAME with your actual GitHub username
git clone https://github.com/YOUR-USERNAME/samsung-genai.git
cd samsung-genai
# Should print your fork URL twice — confirms it worked
git remote -v
After running these commands, keep Git Bash open. You will use it again in Step 6.
6
Pipeline Test — Edit one file and push it
This is the most important step. You will push a real file to GitHub and watch your name appear live on the dashboard. This proves your entire submission pipeline works before any scored lab begins.
Follow these exact steps — do not skip any:
Step 6a — Open File Explorer
Press Windows key + E to open File Explorer.
Step 6b — Navigate to submissions folder
In File Explorer, go to wherever you cloned the repo (Desktop or Documents) → open the samsung-genai folder → open the submissions folder.
You will see a file called checkin.json already there.
Step 6c — Open checkin.json in Notepad
Right-click checkin.json → Open with → Notepad.
(If "Open with" is not visible, click "Show more options" first.)
Step 6d — Edit two fields only
Change only these two lines. Do not touch anything else.
# Replace these two values — nothing else
{
  "student_name": "Your Full Name Here",
  "github": "your-github-username",
  "team": "TBD",
  "status": "ready"
}
Press Ctrl+S to save. Close Notepad.
Step 6e — Open Git Bash inside the samsung-genai folder
In File Explorer, go back up one level so you are inside the samsung-genai folder (not submissions).
Right-click on any empty space inside that folder → click Git Bash Here.
A black Git Bash terminal will open, already pointing to the correct folder.
Step 6f — Run the 3 push commands
git add submissions/checkin.json
git commit -m "Day 1 check-in"
git push origin main
Watch the dashboard on the projector.
Within 30 seconds of your push, your name will appear as a green tick on the live dashboard. That confirms your entire submission pipeline works.
